Thread Divergance

Code Smell - 5

Explanation

Thread divergence in CUDA occurs when threads within a thread block take different paths in the program flow, which can lead to increased execution time and decreased performance.

It is a problem that arises in CUDA programming when different threads in a block follow different execution paths based on conditional statements, resulting in divergent behavior. This can lead to inefficient utilization of hardware resources and can impact performance.
Reduce if, if-else and conditional statemets.

How have we detected?
Detect the kernel code
Detect if there are any conditional loops like if, if-else and conditional statements in kernel code which would cause thread diverange.

Solution to Refactor

To refactor this code smell, you can reduce number of conditional statement.

Try to optimise the number of if, if-else and conditional statements in kernel code.

If possible, remove the conditional statements inside kernel calls and try using that logic before kernel call.

Solution:
Optimise conditional statements inside kernel code.
