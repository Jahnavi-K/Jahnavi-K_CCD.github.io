CCD: CUDA Code Smell catalog

About Tool

CCD: CUDACodesmellDetector is a tool to detect code smells in CUDA programming language.

About CUDA: CUDA is used in parallel programming, to facilitate faster and complex codes which need GPU(Graphical User Interface) assistance along with CPU processing. The first release of NVidia CUDA was in 2007, and since then, there have been tremendous applications built using this interface.

Download the CCD tool from: https://github.com/Abhijeetkumar96/CUDACodeSmell

Instructions to run the Tool:

  1. Run the below command in command prompt or terminal: pip install networkx
  2. Download the DetectorCode folder in this repository
  3. Execute below command to run the tool: python3 tool.py
  4. Select the Cuda project folder for which you want to detect code smells
  5. Click on "Submit" button to see the smells detected.
